Common Commands

  • ls, ls -l, ls -la
  • pwd, whoami, date
  • mkdir test, cd .., cd ~
  • uname -a, df -h, ps aux
  • help to list all supported commands

Tips

  • Use Up/Down arrows for command history
  • Ctrl+L to clear terminal
  • Combine flags like ls -la
